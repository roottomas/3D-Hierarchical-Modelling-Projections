Keys

'h' : Toggle this panel
'0' : Toggle 1/4 views
'1' : Front view
'2' : Left view
'3' : Top view
'4' : 4th view

'9' : Parallel vs Perspective

'e' : Move backward
'w' : Raise cannon
's' : Lower cannon
'a' : Rotate cabin ccw
'd' : Rotate cabin cw
'r' : Reset view params
'←' : Increase theta
'→' : Decrease theta
'↑' : Increase gamma
'↓' : Decrease gamma
'm' : Barrel's minigame
